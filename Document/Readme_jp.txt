■概要 
このプログラムソースコードライブラリの名称は
[st_vba]です。

Windows PC で動作する Excel VBA での実用的な関数ライブラリです。

Excel VBA は Microsoft Excel に含まれる強力なプログラミング言語です。
Excel 上で簡単にプログラムを作成できるために
非常に利用者が多い言語だと思われます。

Excel自体が非常によく使われているアプリケーションなのと
Excel VBA はシンプルでありながら
高機能な言語ですので、開発効率は高いです。

しかし、標準的に備えて欲しい機能なのに
欠けている部分も感じたりします。

そこで Excel VBA での開発をさらに効率のよいものにするために
Excel VBA の標準にはないが汎用的に使える機能を関数としてまとめて
オープンソースのライブラリとして
スタンダードソフトウェアのライブラリとして公開させていただきます。

スタンダードソフトウェアでは
ソースコードライブラリをオープンソースで提供し
開発者の方がより素早く望む機能を実現させる事ができるように
支援させていただきます。

より多くの開発者の方に提供させていただくために
MIT ライセンスとします。

■使用方法

Excel ファイルで VBA プロジェクトを開いてください。
xlsファイルかxlsmファイルを開いた状態で
[Alt+F11キー]でVBAプロジェクトが開きます。

標準モジュールを追加して
モジュール名を st_vba_Base とつけて

st_vba_Base_Module.vba

このファイルの内容を貼り付けてください。

ファイル内容の35行目程度のところにある[■参照設定]にしたがって
いくつかのファイルを参照設定してください。
VBA Editor の[ツール][参照設定]で画面を開くことができます。

基本的にはこれだけで、st_vba の関数群を使用可能です。

他にも vba ファイルはありますが、
応用時に使うことになります。
